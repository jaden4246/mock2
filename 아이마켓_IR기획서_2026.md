# 아이마켓 (KidsTrade)
### 아이들이 직접 사고 파는, 대한민국 최초 아동 전용 C2C 중고거래 플랫폼

---

## 1. Executive Summary

| 항목 | 내용 |
|------|------|
| 서비스명 | 아이마켓 (KidsTrade) |
| 서비스 유형 | 아동 전용 보호자 연동 C2C 중고거래 웹 플랫폼 |
| 타깃 사용자 | 만 6~13세 아동 + 보호자 |
| 론칭 시장 | 수도권 (서울·경기·인천) |
| 수익 모델 | 프리미엄 구독 → 거래 수수료 → B2B 데이터·제휴 (단계적 복합) |
| 투자 요청 | **Pre-Seed 1억 5천만 원** |
| 사용 계획 | MVP 개발 70% · 파일럿 운영 20% · 법무·인프라 10% |

**한 줄 정의**
> 아이들이 직접 등록하고, 부모가 승인하고, 학교·동네 안에서만 거래하는 — 세계 최초 아동 전용 안전 중고거래 플랫폼

**왜 지금인가**
국내 중고거래 시장은 24조 원(2023)으로 성장했지만, 아동을 위한 플랫폼은 단 하나도 없다. 당근마켓·번개장터는 성인 전용 설계이며, 국내외 어디에도 아동 전용 C2C 플랫폼은 존재하지 않는다. 규제 환경(아동 개인정보보호법)이 강화되는 지금이 진입 장벽을 가장 높게 쌓을 수 있는 타이밍이다.

## 2. Problem

### 2-1. 세 가지 핵심 문제

**① 아동은 중고거래 플랫폼을 쓸 수 없다**
당근마켓·번개장터는 만 14세 미만 가입 불가. 아이들이 쓰다 남긴 장난감·책·학용품은 부모 대리 거래나 폐기로 처리된다. 부모 대리 거래는 비체계적이고 사기 위험이 높은 맘카페에 의존한다.

**② 아동 소비재의 낭비가 심각하다**
아동 1인당 연간 완구·교육 소비 금액은 50만 원 이상(2024 통계청). 그러나 대부분의 제품은 1~2년 내 사용 중단 후 방치되거나 폐기된다. 아동 소비재의 재활용률은 성인 소비재 대비 현저히 낮다.

**③ 일상 속 경제교육 기회가 없다**
초등학교 교과에 금융·경제 교육이 포함되어 있으나, 실제 거래 경험을 통한 체험형 교육은 불가능하다. 부모 73%가 자녀의 경제교육에 관심이 있지만 구체적 실천 방법이 없다고 응답한다(한국금융투자자보호재단, 2023).

### 2-2. 기존 대안의 한계

| 대안 | 한계 |
|------|------|
| 당근마켓·번개장터 | 만 14세 미만 이용 불가, 성인과 동일 환경 |
| 맘카페 거래 | 비구조적, 분산, 검증 체계 없음 |
| 오프라인 바자회 | 연 1~2회, 접근성 낮음 |
| 직접 폐기 | 자원 낭비, 환경 부담 |

## 3. Solution

**아이마켓의 핵심 메커니즘: 보호자 이중 승인 구조**

```
아이                   보호자                 플랫폼
 │                       │                      │
 ├─ 물건 등록 요청 ──────►│                      │
 │                       ├─ 내용 확인 ──────────►│
 │                       ├─ 승인 ───────────────►│ 게시 공개
 │                       │                      │
 ├─ 구매 채팅 시작 ───────►│(알림)                │
 ├─ 구매 요청 ────────────►│                      │
 │                       ├─ 거래 장소 확인 ──────►│
 │                       ├─ 승인 ───────────────►│ 거래 완료
```

**세 가지 차별화 설계 원칙**

1. **안전 우선 (Safety-First)** — 아이는 닉네임·캐릭터만 노출, 개인정보 자동 차단, 거래 장소는 학교·커뮤니티센터 등 사전 등록된 안전 장소만 허용
2. **아동 중심 UX (Child-First UX)** — 텍스트 최소화, 이모지·스티커 중심 채팅, 음성 입력 지원, 단계별 가이드 캐릭터
3. **교육 내재화 (Built-in Education)** — 거래 완료 시 씨앗 포인트 적립, 절약 금액·환경 기여 시각화, 등급 뱃지 시스템

## 5. Product

### 5-1. 핵심 기능 4가지

**① 아이 계정 (Child Account)**
- 보호자 계정에 종속된 서브 계정
- 닉네임 + 캐릭터 아바타만 노출 (실명·사진 없음)
- 학교명 또는 동네 인증은 보호자가 수행
- 연령 기반 UI 자동 조정 (저학년/고학년 모드)

**② 물건 등록 & 보호자 승인**
- 카테고리: 장난감 / 도서·학습지 / 문구·미술 / 스포츠·야외 / 의류·신발 / 기타
- 사진 업로드 + 음성 설명 입력 지원
- AI 자동 카테고리 분류 + 적정가격 범위 추천
- 등록 후 보호자 푸시 알림 → 승인 시 자동 게시

**③ 안전 거래 매칭**
- 학교 단위 / 동네 반경 1~3km 이내 매칭
- 채팅: 텍스트 + 스티커만 허용, 전화번호·이메일 자동 마스킹
- 거래 장소: 플랫폼 등록 안전 장소(학교 정문, 도서관, 주민센터) 목록에서만 선택
- 거래 완료 후 보호자 쌍방 확인

**④ 보호자 대시보드**
- 자녀 등록 물품·채팅 내역 열람
- 실시간 알림 (등록 요청 / 채팅 시작 / 거래 제안)
- 월간 거래 내역 + 절약 금액 리포트
- 용돈 계좌 연동 (Phase 2)

### 5-2. 게이미피케이션 (경제교육 내재화)

| 요소 | 내용 |
|------|------|
| 씨앗 포인트 | 거래 완료 시 적립, 추가 등록 슬롯·캐릭터 해금에 사용 |
| 등급 뱃지 | 새싹 → 열매 → 나무 → 숲 (거래 횟수 기반) |
| 절약 시각화 | "이번 달 OOO원 아꼈어요!" 월간 리포트 |
| 환경 기여 | "탄소 OOkg 줄였어요!" — 지속가능 소비 교육 |

### 5-3. 사용자 플로우

```
[가입]
보호자 회원가입 → 자녀 계정 생성 → 학교/동네 인증 완료

[판매]
아이: 사진 촬영 → 음성/텍스트 설명 → 가격 입력 → 등록 요청
보호자: 알림 수신 → 내용 확인 → 승인 → 자동 게시

[구매]
아이: 검색/탐색 → 찜하기 → 채팅 → 구매 요청
보호자: 알림 수신 → 거래 장소 확인 → 승인
아이: 지정 장소에서 직거래 완료
보호자: 거래 완료 확인 → 씨앗 포인트 적립
```

## 6. Business Model

수익 모델은 **3단계 페이즈**로 진화한다. 초기에는 검증된 SaaS형 구독으로 빠르게 수익을 확보하고, 규모가 커지면 거래 수수료와 B2B를 더해 매출 구조를 다각화한다.

### Phase 1 — 구독 (Year 1~2)

| 플랜 | 대상 | 가격 | 혜택 |
|------|------|------|------|
| 무료 | 모든 가입자 | 0원 | 월 등록 3건, 기본 채팅 |
| 프리미엄 | 보호자 | 3,900원/월 | 무제한 등록, 상단 노출, 거래 분석 리포트 |

### Phase 2 — 거래 수수료 추가 (Year 2~3)

- 거래 완료 건당 **3% 수수료** (판매자 부담)
- 안전결제 기능 도입 시 결제 수수료 별도 (카드사 연동)
- 평균 거래 단가 8,000원 × 3% = **240원/건**

### Phase 3 — B2B 확장 (Year 3~)

| 수익원 | 내용 | 단가 |
|--------|------|------|
| 광고 | 교육·완구 브랜드 타겟 배너 | CPM 기반 |
| 학교·기관 제휴 | 바자회·이벤트 연계 수수료 | 건당 계약 |
| B2B 데이터 리포트 | 아동 소비재 트렌드 익명 집계 | 연간 구독 |
| 핀테크 제휴 | 용돈 계좌 연동 파트너십 수수료 | 수익 분배 |

## 7. Go-to-Market Strategy

### 7-1. 수도권 파일럿 전략 (0~6개월)

**채널 1 — 학교 직접 진입**
- 수도권 30개 초등학교 대상 공문·제안서 발송
- 담임교사·학부모회 대상 시범 운영 제안
- 학교별 바자회·환경교육과 연계 이벤트

**채널 2 — 학부모 커뮤니티 바이럴**
- 맘카페(네이버 카페) 상위 50개 타겟 입점
- 학부모 인플루언서(유튜브·인스타) 체험 리뷰
- "아이가 직접 번 첫 용돈" 스토리 캠페인

**채널 3 — ESG·교육 기관 협력**
- 지역아동센터·도서관과 안전 거래 장소 MOU
- 교육청 창의·체험 프로그램 연계 신청

### 7-2. 성장 단계 (7~18개월)

- 수도권 전 지역 서비스 오픈
- 학교 단위 리퍼럴 프로그램 (학교 인증 시 전교생 프리미엄 1개월 무료)
- 완구·교육 브랜드 공동 마케팅 (중고 거래 후 신제품 할인 쿠폰)

### 7-3. 획득 비용 (CAC) 목표

| 구분 | Year 1 | Year 2 | Year 3 |
|------|--------|--------|--------|
| CAC | 1,500원 | 1,000원 | 700원 |
| 주요 채널 | 맘카페·SNS | 리퍼럴 | 자연 유입 |

## 8. Competitive Landscape

### 8-1. 경쟁 포지셔닝

```
                    [아동 특화]
                         ▲
                         │
          아이마켓  ●    │   ← 유일한 포지션
                         │
[중고거래] ◄─────────────┼─────────────► [신품 판매]
                         │
  당근마켓 ●  번개장터 ● │
      중고나라 ●          │
                         ▼
                    [성인 범용]
```

> **핵심**: 아동 특화 + 중고거래 사분면에 경쟁자가 없다. 국내외 모두.

### 8-2. 직접 경쟁사 비교

| 구분 | 당근마켓 | 번개장터 | 아이마켓 |
|------|----------|----------|----------|
| 타깃 | 성인 전체 | 10~30대 | 아동 6~13세 |
| 아동 이용 | 불가 (만 14세 미만 가입 제한) | 불가 | 핵심 사용자 |
| 안전장치 | 신고 기능만 | 신고 기능만 | 보호자 이중 승인 |
| 교육 가치 | 없음 | 없음 | 경제·환경 교육 통합 |
| 지역 신뢰 | 동네 기반 | 없음 | 학교 인증 기반 |
| 규제 대응 | 일반 개인정보보호법 | 일반 | 아동 개인정보보호법 선제 내재화 |

### 8-3. 해외 유사 사례 — 완전한 형태는 없다

| 서비스 | 국가 | 특징 | 아이마켓과의 차이 |
|--------|------|------|-----------------|
| ThredUp Kids | 미국 | 아동 의류 C2B 리셀 | B2C, 아동이 직접 거래 불가 |
| KidsAround | 유럽 | 아동용품 지역 거래 | 보호자 중심 운영, 교육 요소 없음 |
| Vinted (Kids 필터) | 유럽 | 의류 특화 | 성인 플랫폼의 필터 기능, 아동 전용 아님 |

**결론**: 세계 어디에도 아동이 직접 거래하는 전용 C2C 플랫폼은 없다 → **글로벌 블루오션**

### 8-4. 경쟁 우위 (Moat) 5가지

| 우위 요소 | 내용 | 모방 난이도 |
|-----------|------|------------|
| 안전 구조 | 보호자 이중 승인 — 특허 출원 가능 구조 | 높음 |
| 아동 전용 UX | 성인 앱 단순화가 아닌 인지 발달 기반 설계 | 중간 |
| 학교 네트워크 | 학교 채널 파트너십 → 전환 비용 높은 락인 | 높음 |
| 교육적 가치 | 거래 = 교육 콘텐츠 → ESG 투자 매력 | 높음 |
| 규제 선점 | 아동 개인정보보호법 준수 체계 초기 내재화 | 높음 |

연평균 MAU = (연초 + 연말) / 2 기준으로 산출

| 거래 수수료 | — | 6,552 | 16,416 | 31,680 | 54,432 |
| 광고 | 100 | 500 | 1,500 | 3,000 | 5,000 |
| B2B·제휴 | — | — | 500 | 1,500 | 3,000 |
| **매출 합계** | **3,610** | **25,304** | **67,556** | **139,140** | **252,152** |

> 프리미엄 구독: 연평균 MAU × 전환율 × 3,900원 × 12개월
> - Y1: 15,000 × 5% × 3,900 × 12 = 35,100,000원 = 3,510만원
> - Y2: 65,000 × 6% × 3,900 × 12 = 182,520,000원 = 18,252만원
>
> 거래 수수료: 연평균 MAU × 거래 참여율 × 평균 거래가 × 3% × 12개월
> - Y2: 65,000 × 35% × 8,000 × 3% × 12 = 65,520,000원 = 6,552만원
