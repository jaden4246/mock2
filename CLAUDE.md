# 아이마켓 (KidsTrade) 프로젝트

대한민국 최초 아동 전용 C2C 중고거래 플랫폼 창업 기획 프로젝트.
Aalto University MBA 과정 중 실제 창업 준비 + 투자 유치를 동시에 목표로 한다.

---

## 프로젝트 파일 구조

```
기획서/
├── CLAUDE.md                    ← 이 파일
├── 아이마켓_IR기획서_2026.md    ← 메인 기획서 (마크다운 원본)
├── 아이마켓_IR기획서_2026.docx  ← 투자자 배포용 Word 문서
├── 아이마켓_mock.html           ← 인터랙티브 서비스 데모 사이트
└── 아이마켓_서비스기획서.md     ← 초기 초안 (참고용, 더 이상 업데이트하지 않음)
```

**편집 기준 파일은 `아이마켓_IR기획서_2026.md`다.**
내용을 수정할 경우 `.md` → `.docx` 순서로 반영한다.

## 기획서 작성 원칙

- **대상 독자**: 투자자 (VC/엔젤) + 실제 창업 실행 참고 병행
- **문서 스타일**: IR 피치 스타일 — 각 섹션 짧고 임팩트 있게, 숫자·근거 중심
- **언어**: 한국어 (mock 사이트 포함)
- **섹션 순서**: Executive Summary → Problem → Solution → Market → Product → Business Model → GTM → Competition → Financials → Team → The Ask → Risk

### 수정 시 주의사항

- **팀 섹션 창업자 역량** 항목은 아직 미작성 상태. 투자자 미팅 전 반드시 업데이트.
- 재무 수치는 내부 일관성을 반드시 검증한다 (연평균 MAU 기준으로 매출 역산).
- BEP: **Year 4 중반** (MAU 35만, 프리미엄 전환율 8% 기준) — 수치 변경 시 손익 요약표와 함께 업데이트.

## mock 사이트 구성

파일: `아이마켓_mock.html` — 단일 파일, 외부 의존성 없음 (오프라인 실행 가능)

**랜딩 페이지 섹션 순서**
1. Hero (서비스 카피 + 통계)
2. Problem (3가지 Pain Point)
3. Solution (6단계 거래 플로우)
4. Features (핵심 기능 4가지)
5. Market (TAM/SAM/SOM 시각화)
6. Business Model (Phase 1→2→3)
7. Competition (포지셔닝 맵)
8. CTA (투자 문의)

**앱 프로토타입 (인터랙티브)**
- 아이 화면 3종: 홈 → 상품 상세 → 채팅
- 보호자 화면 3종: 대시보드 → 승인 → 거래 내역
- 승인 버튼 클릭 시 피드백 애니메이션 동작

## docx 변환 방법

Node.js 없는 환경이므로 `python-docx` 라이브러리를 사용한다.

```bash
pip3 install python-docx
python3 convert_to_docx.py   # 필요 시 스크립트 재작성
```

변환 스크립트는 일회성으로 삭제했으므로, 재변환이 필요하면 Claude에게 요청한다.
