🚫

성인 중심 플랫폼의 배제

국내 모든 중고거래 플랫폼은 만 14세 미만 가입이 불가합니다. 아이들이 쓰다 남긴 물건은 부모 대리 거래나 폐기에 의존할 수밖에 없습니다.

📊 국내 아동 특화 C2C 플랫폼: 0개
🗑️

아동 소비재의 막대한 낭비

아동 1인당 연간 완구·교육 소비는 50만 원 이상. 대부분의 제품이 1~2년 내 사용 중단 후 방치되거나 폐기됩니다. 재활용률이 현저히 낮습니다.

💰 연간 아동 완구 시장: 2조 4천억 원
📖

체험형 경제교육의 부재

부모 73%가 자녀 경제교육에 관심이 있지만, 실제 거래 경험을 통한 체험형 교육 기회는 일상에서 거의 불가능합니다.

📈 금융교육 관심 학부모: 73% (2023)

Features

아이마켓만의
핵심 기능

아동의 인지 발달 수준에 맞게 설계된, 성인 앱의 단순화가 아닌 완전히 새로운 경험.

🔐

아이 전용 계정

보호자 계정에 연동된 서브 계정으로 닉네임·캐릭터만 노출. 개인정보는 절대 공개되지 않습니다.

  • 실명·사진 비공개
  • 학교 인증은 보호자가 진행
  • 저학년/고학년 UI 자동 조정
📍

학교·동네 기반 매칭

같은 학교 또는 반경 1~3km 이내 아이들끼리만 거래. 낯선 성인과의 접촉을 원천 차단합니다.

  • 학교 단위 우선 매칭
  • 안전 거래 장소 목록 제공
  • 거래 장소 보호자 확인 필수
👨‍👩‍👧

보호자 대시보드

자녀의 등록 물품, 채팅 내역, 거래 현황을 한눈에. 모든 거래는 보호자 승인 없이 진행되지 않습니다.

  • 실시간 알림 및 1-tap 승인
  • 월간 거래 내역 리포트
  • 절약 금액·환경 기여 시각화
🎮

경제교육 게이미피케이션

거래 자체가 경제 교육이 됩니다. 씨앗 포인트, 등급 배지, 절약 시각화로 재미있게 배웁니다.

  • 씨앗 포인트 → 새싹 → 나무
  • "이번 달 OOO원 아꼈어요!"
  • "탄소 OOkg 줄였어요!" 환경 교육
